4) HCF AND LCM
--------------
START
  |
Input A, B
  |
i = 1, HCF = 1
  |
Is i <= A AND i <= B ?
  |----No----> Calculate LCM
  |
If A % i == 0 AND B % i == 0
  |
HCF = i
  |
i = i + 1
  |
(Repeat Loop)

LCM = (A * B) / HCF
Print HCF, LCM
END


5) SUM UNTIL USER ENTERS 'x'
---------------------------
START
  |
Sum = 0
  |
Input Value
  |
Is Value == 'x' ?
  |----Yes----> Print Sum -> END
  |
Sum = Sum + Value
  |
Input Next Value
  |
(Repeat Loop)
